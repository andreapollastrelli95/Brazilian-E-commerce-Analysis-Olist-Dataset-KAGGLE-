# Brazilian-E-commerce-Analysis-Olist-Dataset-KAGGLE-
Questo progetto analizza il Brazilian E‑commerce Public Dataset di Olist (Kaggle), uno dei dataset più completi disponibili sul commercio elettronico. L’obiettivo è comprendere il comportamento dei clienti, le performance di vendita, la distribuzione geografica dei ricavi e le dinamiche temporali degli ordini.
